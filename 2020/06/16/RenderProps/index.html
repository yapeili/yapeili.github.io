渲染属性(Render Props)

用于使用一个值为函数的 prop 在 React 组件之间的代码共享(数据共享)

带有渲染属性(Render Props)的组件需要一个返回 React 元素并调用它的函数，而不是实现自己的渲染逻辑。

组件是React 主要的代码复用单元，但如何共享状态或一个组件的行为封装到其他需要相同状态的组件中并不是很明了
